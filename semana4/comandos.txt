Primero instalar:
npm install prisma --save-dev
npm install @prisma/client

SI CAMBIAMOS EL SCHEMA

npx prisma generate

RESETEAR MIGRACIONES

- Importante cuando borramos o movemos una carpeta, cambiamos rutas o reescribimos

npx prisma migrate reset -- Borra todos los datos anteriores

PONER NOMBRE A CARPETA DE MIGRACIONES

npx prisma migrate dev --name add-student-course


VERSION ESTABLE:

npm install prisma@6.19.0 --save-dev
npm install @prisma/client@6.19.0
npx prisma --version
